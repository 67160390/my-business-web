# MIXUE Ice Cream & Tea Web

นี่คือเว็บไซต์ธุรกิจตัวอย่างสำหรับ MIXUE Ice Cream & Tea

## Features

- หน้าแรก (Homepage) รองรับการแสดงผลทุกอุปกรณ์
- หน้า About เล่าเรื่องราวของแบรนด์และทีมงาน
- หน้า Services และ Contact สำหรับบริการและติดต่อ

## Project Structure

```
my-business-web/
├── index.html
├── about.html
├── contact.html
├── services.html
├── images/
│   └── ... (โลโก้ ผลิตภัณฑ์ และรูปทีมงาน)
├── css/
│   └── styles.css
└── README.md
```
